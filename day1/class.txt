코딩 VS 프로그래밍 
코딩 : 컴퓨터에게 명령을 작성하는 일 자체를 의미 (코드를 작성하는 일)
프로그래밍 : 코드 작성, 분석 및 구현, 디버깅, 컴파일, 테스트

웹페이지 vs 웹사이트
웹페이지 : Hyper Text로 작성된 문서
웹사이트 : URL을 기반으로 인터넷에서 서비스되는 웹페이지들의 집합
URI -> URL(위치에 의해서 리소스 구분), URN(이름에 의해서 리소스 구분)

URL(Uniform Resource Locater) : 네트워크 상에서 자원의 위치를 알려주는 규칙

1. 클라이언트 / 서버 구조에서 클라이언트는 요청(request) 서버는 응답(response)
2. 브라우저는 하이퍼 텍스트를 해석하여 보여주는 클라이언트
3. HTML은 하이퍼텍스트를 표현하는 마크업 언어 -> 웹페이지를 구조적으로 기술
    대표적인 기능 : Hyper Link - 텍스트를 클릭하면 링크된 다른 웹 문서로 이동하는 문서 연결 기능
4. 하이퍼텍스트는 정해진 순서없이 참조를 통하여 다른 문서에 접근
5. 웹페이지는 웹에 있는 HTML로 작성된 문서
6. 웹 사이트는 URL 기반으로 인터넷에서 서비스되는 웹페이지들의 집합
7.WWW(World Wide Web) : 웹(www3c.org)
