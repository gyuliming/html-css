(1) DOM은 어떤 말의 약자인가요?
1. Document Object Model

(2) HTML 문서가 완전히 로드된 후에(DOM이 완전히 구성된 후에) 자바스크립트 코드를 실행할 때 갖아 적절한 이벤트는 무엇인가요?
2. DOMContentLoaded

1. onload => 브라우저가 모든 리소스까지 전부 불러온 후 실행
3. onready => jQuery 개념

(3) 자바스크립트로 스타일을 직접 변경하려면 어떤 속성을 사용해야 하나요?
1. 요소.style

2. 요소.css => HTML 요소의 인라인 속성 변경할 때 사용
3. 요소.stylesheet => 존재하지 않음

(4) <ul> 요소 내부에 새로운 <li> 요소를 추가할 때 자바스크립트에서 사용하는 메서드는 무엇인가요?
1. appendChild()

2. append() => 노드뿐만아니라 문자열까지 가능
3. insert() => 특정 위치에 노드 삽입

(5) DOM에서 요소의 텍스트 내용을 변경할 때 사용할 수 없는 속성은 무엇인가요?
3. textHTML

1. textContent => 요소의 텍스트 내용 변경
2. innerText => 화면에 보이는 텍스트 변경
